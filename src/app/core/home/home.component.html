Welcome to the Recipe Book

Here a summary of the application functionality.

  • This application will load a default set of recipes every time it is opened.
  • You my view the Recipes and the Shopping List without logging in.
  • Use the Recipes menu option to see your list of recipes.
  • Use the Shopping List menu option to see your list of ingredients with quantities.
  • To Save and Retrieve the list of recipes from the server you must login.
  • (Please take note: All users access the same instance of the data on the server. In other words, there is only one copy for all users.)
  • Recipes options:
    1. Select the recipe to see its details.
    2. Add a new recipe.
